# 🚀 Overview

EduInsight is a modern educational analytics platform that enables administrators, teachers, and students to monitor academic performance through interactive dashboards, AI-powered insights, and real-time analytics.

The platform combines data visualization, role-based authentication, and Google's Generative AI to transform raw educational data into actionable recommendations.
